Split shell command line into a list of arguments. Aims to emulate {\ttfamily bash} and friends.


\begin{DoxyItemize}
\item Arguments are delimited with whitespace
\item Extra whitespace at the beginning and end and between arguments will be ignored
\item Text can be \char`\"{}double\char`\"{} or \textquotesingle{}single\textquotesingle{} quoted
\item The backslash {\ttfamily \textbackslash{}} is used as escape character
\begin{DoxyItemize}
\item Outside quotes, any character can be escaped
\item Within double quotes, only escape {\ttfamily \char`\"{} and backslashes before a \textbackslash{}c \char`\"{}} or another backslash
\item Within single quotes, no escaping is possible and no special interpretation takes place
\end{DoxyItemize}
